# Libero

A WIP component library designed to be as frictionless as possible in terms of dx, ux, a11y and configurability.

## Design principles
1. Frictionless experience: Dx, Ux, A11y, Documentation
2. Customizable: Themes, styles
3. Opinionated: Production ready components, hooks and utilities

## Roadmap
1. ThemeProvider & Sx like styling
2. Some basic components & layouts
3. More advanced components
